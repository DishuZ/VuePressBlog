卡特兰数 (opens new window)：

  • 时间复杂度为 O(n)O(n)O(n) 的通项公式：Cn+1=Cn⋅(4n+2)n+2,C0=1C_{n+1} = \frac{C_n \cdot (4n+2)}{n+2}, C_0 = 1Cn+1​=n+2Cn​⋅(4n+2)​,C0​=1
private int catalan(int n) {
  if (n <= 1) return 1;

  long ans = 1;
  for (int i = 0; i < n; ++ i ) ans = ans * (4 * i + 2) / i + 2;
  
  return (int)ans;
}

质数

  • 质数和合数是针对所有大于1的 “自然数” 来定义的（所有小于等于1的数都不是质数）
  • 所有小于等于1的整数既不是质数也不是合数
  • 质数的判定：试除法 ○ d∣nd|nd∣n 代表的含义是整数 ddd 能整除 nnn (这里的“ ∣|∣ ”代表整除) ○ 一个合数的约数总是成对出现的，如果 d∣nd|nd∣n，那么必然有 (nd)∣n(\frac{n}{d})|n(dn​)∣n 。因此我们判断一个数是否为质数的时候，只需要判断较小的那一个数能否整除 nnn 就行了，即只需枚举 d≤(nd)d \le (\frac{n}{d})d≤(dn​) 就行了

质因数(素因数或质因子)：在数论里是指能整除给定正整数的质数

  • 除了1以外，两个没有其他共同质因子的正整数称为互质。因为1没有质因子，1与任何正整数（包括1本身）都互质
  • 每个 合数都可以写成几个质数相乘的形式，这几个质数就都叫做这个合数的质因数

long qmi(long a, long b, long p) {
  long res = 1;
  while (b != 0) {
    if ((b & 1) == 1) res = res * a % p;
    a = a * a % p;
    b >>= 1;
  }
  return res;
}
